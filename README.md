# Simple Python ETL

A simple implementation of an Extract, Transform, Load (ETL) pipeline using Python and SQLite.

## Overview

This repository demonstrates a lightweight ETL system. It is designed to extract data from a source, apply basic transformations, and load the processed data into a local SQLite database (currently consuming data from a internal table created within the code).

## Technologies Used

* **Python 3.12.1 (Pandas lib)**
* **SQLite3 3.45.3**



**Functional, but under development (feel free to help or improve the code).**
